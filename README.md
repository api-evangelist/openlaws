# OpenLaws (openlaws)

OpenLaws provides programmatic access to law data including statutes, regulations, and case law across U.S. jurisdictions. The API supports keyword and hybrid search, jurisdiction and court filtering, citation identification and validation, historical version queries with redline comparisons, and citation lookup with mapping to authoritative government sources.

## APIs

### OpenLaws

OpenLaws API exposes search across statutes, regulations, and case law with BM25 or hybrid retrieval; jurisdiction and court filtering; legal citation parsing and validation; historical version queries and redline comparisons; and citation-to-government-source mapping for use in research and AI applications.
